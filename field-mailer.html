Field Mailer - Promotion

A custom modal design to get the subscribers to the field.

Problem Statement :

How to add a feature so that the users can actually subscribe to a particular section. In this case, we are considering the field. By subscribing to the field the users will get the updates via a mail regarding the sports section.

Team :

3 people (including PM, developer and Me)

Technologies Used :

Sketch, HTML, Sass

Role :

Designed a custom modal that will get the basic information from the user i.e. Email and give them a confirmation message in the same modal window without letting a single pixel moved. Also, the design should be responsive so that the mobile users can also get the best experience.
